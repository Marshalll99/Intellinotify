## Introduction

Intellinotify is designed to deliver a robust notification system with a modern, responsive front-end. The project is built using:
- **Django**: as the backend framework.
- **Tailwind CSS**: for building a beautiful and responsive user interface.
- **django-tailwind**: for seamless integration of Tailwind CSS into Django.

The project is structured into multiple apps:
- **core**: Contains the main project settings, URL configuration, and WSGI/ASGI files.
- **data_engine**: Handles data processing and backend logic, including models (like notifications).
- **notifier**: Reserved for notification-specific functionalities (future work).
- **pages**: Contains the frontend pages (templates and views) such as Home and Notifications.
- **theme**: Manages Tailwind CSS configuration via django-tailwind.

## Setup Instructions

### Prerequisites

- **Python**: 3.12 (or a compatible version, e.g., 3.11)
- **Node.js and npm**: for Tailwind CSS (download from [nodejs.org](https://nodejs.org/))
- **Git**: for version control

### Creating a Virtual Environment

In your project’s root directory, run:
```bash
python3 -m venv .venv
```

Activate the virtual environment:

- **macOS/Linux:**
  ```bash
  source .venv/bin/activate
  ```
- **Windows:**
  ```bash
  .venv\Scripts\activate
  ```

### Installing Dependencies

With your virtual environment activated, install Django, django-tailwind, and other required packages:
```bash
pip install django django-tailwind beautifulsoup4 requests
```

Generate a requirements file:
```bash
pip freeze > requirements.txt
```
A sample **requirements.txt** might include:
```
asgiref==3.8.1
beautifulsoup4==4.13.3
Django==5.1.6
django-tailwind==3.8.0
requests==2.32.3
# ... other packages
```

### Setting Up the Django Project

If you haven't created the project yet, run:
```bash
django-admin startproject core .
```

### Integrating Tailwind CSS

1. **Install django-tailwind (if not already installed):**
   ```bash
   pip install django-tailwind
   ```
2. **Update `core/settings.py`:**
   ```python
   INSTALLED_APPS = [
       'tailwind',
       'theme',  # Tailwind theme app (see next step)
       'pages',
       'data_engine',
       'notifier',
       'django.contrib.admin',
       'django.contrib.auth',
       'django.contrib.contenttypes',
       'django.contrib.sessions',
       'django.contrib.messages',
       'django.contrib.staticfiles',
   ]
   TAILWIND_APP_NAME = 'theme'
   ```
3. **Create the Tailwind Theme App:**
   - Temporarily comment out `'theme'` in **INSTALLED_APPS** if necessary.
   - Run:
     ```bash
     python manage.py tailwind init
     ```
   - When prompted (e.g., `[1/1] app_name (theme):`), press **Enter** to accept the default name.
   - Re-add `'theme'` to **INSTALLED_APPS** if you commented it out.
4. **Install Tailwind CSS dependencies:**
   ```bash
   python manage.py tailwind install
   ```
5. **Start the Tailwind watcher (for live reloading during development):**
   ```bash
   python manage.py tailwind start
   ```

## Running the Project

Start the Django development server:
```bash
python manage.py runserver
```
Access your site at [http://127.0.0.1:8000/](http://127.0.0.1:8000/).

## Working with the Project

### Database Migrations

After modifying models, run:
```bash
python manage.py makemigrations
python manage.py migrate
```

### Creating a Superuser

To create an admin account:
```bash
python manage.py createsuperuser
```

### Additional Commands

- **Collect Static Files (for production):**
  ```bash
  python manage.py collectstatic
  ```
- **Run Tests:**
  ```bash
  python manage.py test
  ```
